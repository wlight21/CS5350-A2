- running experiments
	- unzip Experiments.zip
	- run run.sh on CADE with ./run.sh
	- this shell script simply runs main.py

- output overview
	- running run.sh will print outputs into organized text files
	- the text files are named after the experiment whose output they contain

- output breakdown
	- mode_label.txt
		- experiment output for the most common label model
	- StandardPerceptron.txt
		- a-e experiment output for the standard perceptron variant
	- DecayPerceptron.txt
		- a-e experiment output for the decay perceptron variant
	- MarginPerceptron.txt
		- a-e experiment output for the margin perceptron variant
	- AveragePerceptron.txt
		- a-e experiment output for the average perceptron variant
